
# Simple Text Based Calculator

A simple arithmetic text based calculator.
The calculator using Shunting yard algorithm to convert the infix expression to prefix.




## Expressions for Example

```javascript
Input
i = 0
j = ++i
x = i++ + 5
y = 5 + 3 * 10
i += y


Output
(i=37,j=1,x=6,y=35)
}
```

## Running Tests

  mvn test
